# My Custom Device Integration

This integration allows you to connect your My Custom Device to Home Assistant.

## Features

- Easy configuration through the UI
- Real-time sensor data
- Automatic updates

## Configuration

Add the integration through the Home Assistant UI:
1. Go to Settings -> Devices & Services
2. Click "+ Add Integration"
3. Search for "My Custom Device"
4. Enter your device details

## Usage

Once configured, the integration will create sensor entities for your device. You can use these in automations, dashboards, and scripts.
